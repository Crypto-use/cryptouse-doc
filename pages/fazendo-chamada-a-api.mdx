# Fazendo uma chamada a api

Utilizando o AppID que você obteve na seção anterior, você pode fazer uma chamada a api

```sh
curl -X POST https://api.cryptouse.com.br/api/order/create \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer <APPID>' \
-d '{
  value: 10,
}
```

A resposta será algo parecido com isso

**Obs:** No payload o value está como 10, a resposta aparece 1000, pois usamos centavos para medida, 1 real = 100 centavos.
